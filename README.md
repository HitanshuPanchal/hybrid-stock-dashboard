## ðŸ“Š Hybrid Trading Logic

The final signal is calculated using:

price_change = (predicted_price - actual_price) / actual_price * 100
combined_score = 0.7 * price_change + 0.3 * sentiment_probability


| Combined Score | Trading Signal |
|---------------|---------------|
| > 0.2         | BUY           |
| < -0.2        | SELL          |
| Otherwise     | HOLD          |

This prevents over-trading due to small fluctuations.
